JetBot Delivery — Autonomous Navigation from Room 203 to Lobby BRAIL

Proyek ini mengintegrasikan hasil Practical Work 1–3 (Setup, Map Generation, dan Navigation) menjadi satu sistem robot pengantar barang otonom.
Robot melakukan navigasi dari Ruang 203 ke Lobby BRAIL (Lantai 2) menggunakan ROS2, Nav2, dan waypoint navigation.

Pada implementasi ini, robot melakukan simulasi aktivitas pengambilan dan pengantaran barang menggunakan bunyi buzzer:

1x beep saat pick di Ruang 203

2x beep saat place di Lobby BRAIL

Lengan robot tidak digunakan, dan sesuai dengan sistem demonstrasi, hanya navigasi yang diaktifkan.
